# Extract Log-Likelihood from gkwreg2 Object

Extract Log-Likelihood from gkwreg2 Object

## Usage

``` r
# S3 method for class 'gkwreg2'
logLik(object, ...)
```

## Arguments

- object:

  A fitted `gkwreg2` object.

- ...:

  Currently ignored.

## Value

A `logLik` object with attributes `df` (degrees of freedom) and `nobs`
(number of observations).
