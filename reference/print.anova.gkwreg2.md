# Print ANOVA Table

Print ANOVA Table

## Usage

``` r
# S3 method for class 'anova.gkwreg2'
print(
  x,
  digits = max(3, getOption("digits") - 3),
  signif.stars = getOption("show.signif.stars"),
  ...
)
```

## Arguments

- x:

  An `anova.gkwreg2` object.

- digits:

  Number of significant digits.

- signif.stars:

  Logical: print significance stars?

- ...:

  Currently ignored.
