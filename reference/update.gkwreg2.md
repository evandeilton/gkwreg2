# Update a gkwreg2 Model

Update a gkwreg2 Model

## Usage

``` r
# S3 method for class 'gkwreg2'
update(object, formula., ..., evaluate = TRUE)
```

## Arguments

- object:

  A fitted `gkwreg2` object.

- formula.:

  Changes to the formula.

- ...:

  Additional arguments to `gkwreg2`.

- evaluate:

  If `TRUE`, evaluate the updated call.

## Value

Updated `gkwreg2` object or call.
