# Print gkwreg2 Object

Displays a concise summary of a fitted `gkwreg2` model.

## Usage

``` r
# S3 method for class 'gkwreg2'
print(x, digits = max(3, getOption("digits") - 3), ...)
```

## Arguments

- x:

  A fitted `gkwreg2` object.

- digits:

  Number of significant digits for printing.

- ...:

  Currently ignored.

## Value

Invisibly returns `x`.
