# Akaike Information Criterion for gkwreg2 Object

Akaike Information Criterion for gkwreg2 Object

## Usage

``` r
# S3 method for class 'gkwreg2'
AIC(object, ..., k = 2)
```

## Arguments

- object:

  A fitted `gkwreg2` object.

- ...:

  Additional `gkwreg2` objects for comparison.

- k:

  Penalty parameter (default 2 for AIC).

## Value
