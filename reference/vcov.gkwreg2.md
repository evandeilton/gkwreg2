# Extract Variance-Covariance Matrix from gkwreg2 Object

Extract Variance-Covariance Matrix from gkwreg2 Object

## Usage

``` r
# S3 method for class 'gkwreg2'
vcov(object, ...)
```

## Arguments

- object:

  A fitted `gkwreg2` object.

- ...:

  Currently ignored.

## Value

The variance-covariance matrix of the estimated coefficients, or `NULL`
if not available.
